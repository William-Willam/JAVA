# ☕ Lógica de Programação - Algoritmos usando Java

Repositório criado para armazenar as aulas, exercícios e exemplos desenvolvidos durante os estudos de **Lógica de Programação utilizando Java**.

## 📚 Nível

**Iniciante**

Este material é destinado para quem está começando na programação e deseja aprender os conceitos fundamentais utilizando a linguagem Java.

- Operadores

### 4. Estrutura Condicional

- Operador ternário
- Exercícios práticos

### 5. Estruturas Repetitivas
- Laço `while`
- Laço `do-while`
- Laço `for`

- Contadores e acumuladores
- Exercícios práticos

Desenvolver a base necessária para a criação de algoritmos e resolução de problemas utilizando a linguagem Java, preparando o estudante para conteúdos mais avançados de programação orientada a objetos.
